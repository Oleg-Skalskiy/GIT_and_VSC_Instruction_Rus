## Скачиваем и устанавливаем GIT и Visual Studio Code

* скачиваем GIT по ссылкам ниже для своей операционной системы и устанавливаем.

    * [Windows](https://git-scm.com/download/win)
    * [Linux](https://git-scm.com/download/linux)
    * [Mac](https://git-scm.com/download/mac)

* Рекомендуем ознакомиться с инструкцией по установке и настройки GIT по ссылке ниже.

    * [FAQ](https://git-scm.com/book/ru/v2/)

* скачиваем Visual Studio Code по ссылкам ниже для своей операционной системы и устанавливаем.

    * [Windows, Linux or Mac](https://code.visualstudio.com/docs/supporting/faq)

* Рекомендуем ознакомится с инструкцией по установке и настройки Visual Studio Code по ссылке ниже.

    * [FAQ](https://code.visualstudio.com/docs/supporting/faq)

* После установки выше перечисленного запускаем Visual Studio Code и немного настроим его.

    * Нажимаем последовательно кнопки на клавиатуре CTRL+P и вызовим командную строку для поска и ввода параметров. Ниже скрин как выглядит командная строка параметров.

## Познакомимся с основными (базовыми) командами GIT

Для начала запустим nthvbyfk

1. Как задать имя пользователя и адрес электронной почты

    `git config --global user.name "Ваше имя"`

    `git config --global user.email "Ваш Email"`

В кавычках подставляем ваши данные и отправляем их в терминале
